## 核心逻辑

### 正确流程
1. **注册**: 创建账号 → 验证验证码 → 成功 → 跳转登录
2. **登录**: 验证密码 → 检查个人空间 → 无则创建 → 跳转工作台

### 验证码管理
- 发送时：存储到内存 + 数据库审计
- 验证时：从内存验证
- 注册成功后：手动删除
